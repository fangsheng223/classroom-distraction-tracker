# Derived Release

This directory contains the privacy-aware derived annotation package included with the public release.

## Files

- `derived_status_boxes.csv`: per-frame box annotations, IDs, and status labels
- `manifest.json`: schema and metadata for the derived package

## Intended use

- evaluation-level reproducibility
- comparison of tracking / status methods without redistributing raw classroom videos
